# Project MITAI (三田井) - Alpha Build

![Version](https://img.shields.io/badge/Version-0.1.0--Alpha-red)
![Status](https://img.shields.io/badge/Status-In--Development-cyan)

**Project MITAI** is an experimental web project currently in the early Alpha phase. It focuses on unique UI/UX design, interactive elements, and a stylized audiovisual experience.

**Project MITAI** is a combination of many games.

## 🚀 How to launch the project?

### 💻 On PC (Windows)
Run the `project_mitai.cmd` file located in the root folder. 
* The script will perform an automatic system file verification.
* If all modules are present, the project will open in your default browser.
* If files are missing, a `errors/M-0.vbs` system alert will be triggered.

### 📱 On Mobile (Android/iOS)
Open the `index.html` (Launcher) in your mobile browser.
* For the best experience, use the **"Add to Home Screen"** option in your browser menu to run it in full-screen mode.
* The launcher will verify module integrity and ensure a safe boot sequence.

## 🚀 Quick Start
1. Open `index.html` in any web browser.
2. Click "INITIALIZE PROJECT" to start the boot sequence.
3. If files are missing, the system will trigger a simulated VBS error.
